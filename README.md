# how-to-install-nodejs-ubuntu
how to install nodejs ubuntu


## install nvm
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
nvm --version
```

## install nodejs
```
nvm install 14.4.0
node --version
npm --version
```


## upgrade node js
```
curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -
sudo apt-get install -y nodejs
```
